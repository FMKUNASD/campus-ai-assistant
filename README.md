# 校园智行助手

本项目是浙江树人学院《人工智能》课程大作业。项目使用前端技术实现一个小型智慧校园交互系统，可在浏览器中运行，包含校园问答、学习计划生成、校园能耗数据分析三个核心功能。

## 功能介绍

1. 校园问答：学生输入校园生活或学习问题，系统调用大模型生成回答。
2. 学习计划生成：根据课程、空闲时间和目标，自动生成一日学习安排。
3. 能耗数据分析：对内置楼宇用电数据进行统计，并生成节能建议。

## 运行方式

直接双击 `index.html` 即可打开。也可以在本目录启动任意静态服务器后访问页面。

## 大模型接口配置

页面左侧提供三种模式：

- 离线演示：无需接口，适合课堂展示。
- Ollama 本地模型：默认地址 `http://localhost:11434`，模型示例 `qwen2.5:7b`。
- OpenAI 兼容接口：填写 `/v1/chat/completions` 兼容服务地址、模型名称和 API Key。
